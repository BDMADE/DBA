# DBA

A cloud application of Dhaka Bar association made through laravel 4.2 framework and twitter-bootstrap framework.<br/>
It has five sections.<br/>
1. Accounts </br>
2. Members </br>
3. Library </br>
4. Users </br>
5. Voter lists </br>
